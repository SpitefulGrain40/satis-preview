Underline treatment — making it feel typographic

The rounded bar reads as a UI element. The fixes below try to make the underline feel like it was drawn with the same pen as the letterforms — taking cues from the slab serifs of the capital I, the em-dash, and the stroke contrast of Playfair itself.

SATIS

B0 — Rounded (current)

For reference. The pill-shape reads as UI — like a notification dot or button. Doesn't share a visual language with the serifs.

SATIS

B1 — Flat, sharp corners

Smallest possible change. Removing the radius alone makes it read more like a typographic stroke. Honest and quiet.

SATIS

B2 — Slab-serif end caps (mini "I")

The bar mirrors the structure of a capital I: vertical stem caps at each end of a horizontal stroke. Reads as a typographic element. Has the most "considered" feel.

SATIS —

B3 — Em-dash glyph (Playfair)

Uses the actual em-dash character from Playfair Display. Drawn by the same designer as the letterforms — guaranteed to share a visual language. Width is fixed by the font.

SATIS

B4 — Tapered to points

Thicker in the middle, tapering to fine points. Mirrors Playfair's high-contrast strokes. Most expressive — and the most likely to look fussy at small sizes.

SATIS

B5 — Hairline (1px, sharp)

Very thin, very sharp. Echoes Playfair's hairline strokes. Risk: disappears at icon size, can look like a div border.

Hairline — tuned at each scale

The hairline weight scales with the typography. At hero size the underline should match the actual hairline strokes of Playfair's S; at icon size it gets slightly thicker to survive. This keeps the bar reading as a typographic element across the whole asset set.

My honest take: Pattern B (top indicator + filled icon) is the right pick — it solves the luminance-collision problem by adding a structural cue, stays inside the sage palette so the brand carries through, and matches the platform pattern users already expect from native apps. A is also good if you want a softer "tablet-ish" feel. D is intriguing but introduces a "nav uses honey" rule that's harder to remember and easier to break later. C is the most minimal but probably *too* subtle for a primary navigation control.

My read after building this: The brand-doc shortlist (Cormorant / Playfair / Baskerville) was framed around editorial elegance, but an S-mark inside a 56px container is a different problem from a hero headline. The Garamonds at 600–700 are caught between worlds. The strongest options for what you actually need are C (Cormorant 700) — retains the refined feel the brand doc asked for, and the heavier weight gives it a fighting chance at icon size — or F (EB Garamond) — slightly more workhorse character but reads as historic/considered rather than systemy. If neither of those land, A (Georgia) is the safe ground you already trust from Hale. Playfair (D) and Lora (H) feel like the wrong vibe for "Satis" — too editorial / too soft respectively.
